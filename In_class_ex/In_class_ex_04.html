Getting Started

Setting up R packages

To start with, ggstatplot and tidyverse packages will be launched using library().

packages = c('ggstatsplot', 'tidyverse')
for (p in packages){
  if(!require(p, character.only = T)){
    install.packages(p)
  }
}
